# Fraud Detection Frontend

## Overview

Fraud Detection Frontend is a modern profile analytics and investigation dashboard built to enhance fraud investigation capabilities. It integrates AI-powered investigative chat, real-time transaction review (approve/block), and client-level blacklisting functionality. The system enables investigators to quickly identify suspicious patterns, perform deep-dive analysis through LLM interactions, and execute decisive actions on flagged profiles and transactions.

## Getting Started

### Installation

To begin, install the required dependencies using the following command:

```bash
pnpm i
```

# Development Server

After installing the dependencies run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
